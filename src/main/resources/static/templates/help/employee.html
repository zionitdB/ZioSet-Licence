ADP Guides
Here is a checklist for your ADP account, please ensure you go through these pointers which will help get your account ready for ADP

  1. Ensure you have added all employees
    - Admin Login>Master>Employee >Add One

    - Bulk Upload employee list

  2. Ensure you have added all User
    - Admin Login>Master>User >Add One

  3. Ensure you have added all Device
    - Admin Login>Master>Device >Add One


  4. Ensure you have added all Tag
    -Admin Login>Master>Tag Upload >Add One

    - Bulk Upload Tag list

  5. Ensure you have added all Asset
    - Admin Login>Master>Asset >Add One

    - Bulk Upload Asset list

  6. Ensure you have added all Mail
    - Admin Login>Master>Mail Manager >Add One
