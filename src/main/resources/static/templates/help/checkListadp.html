ADP Checklist
Checklist before you execute your ADP

  1. Check list for RFID Reader

    1. Ensure RFID reader is connected with Network
    2. Ensure the power supply is working
    3. Ensure the RFID reader configuration is correct
    4. Ensure the RFID reader is set at client mode
    5. Ensure RFID reader is sending data to server

  2. Check list for RFID Asset Tag

    1. Ensure that RFID tag is properly Pasted on the asset with Hostname printed on tag
    2. Ensure that RFID tag is mapped with asset serial number as per hostname printed on tag

  3. User Role

    We have develop the below user role to manage there operation . Each role has the different permission .

    1. Super admin/Admin : All control with both location with data Delete access
    2. Manager : Limited control for both location ( eg: Delete operation restricted )
    3. User : Limited access for user as per location.
